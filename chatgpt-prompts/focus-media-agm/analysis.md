# 内容分析 - 分众传媒股东大会

## 主题
分众传媒年度股东大会，核心看点是超大规模现金分红（49.10亿）与两项议案的高支持度对比。

## 关键信息点
- **参会规模**: 现场58人 + 网络，合计2344人，代表股份46.36亿股（占32.1%）
- **表决氛围**: 所有议案高票通过，无一票否决，过程顺畅
- **分红方案（核心亮点）**:
  - 每10股派1.90元，本次派现27.44亿元
  - 全年现金分红总额 **49.10亿元**
  - 占2025年归母净利润 **166.66%**（动用了以往未分配利润）
  - 在净利润同比下滑超四成背景下仍维持高力度分红
- **议案同意率对比**:
  - 利润分配预案：**99.78%**（中小股东 99.15%，最高支持度）
  - 理财产品议案：**77.61%**（争议较大，相对谨慎）

## 情绪氛围
慷慨、自信、对比鲜明。分红部分体现管理层对现金流的信心与股东回报诚意；理财议案部分体现谨慎与分歧。

## 核心视觉元素
- **电梯广告屏**（分众传媒标志性业务，最具辨识度的视觉锚点）
- 股东大会现场 / 投票箱 / 举手表决
- 大金币堆或钱袋代表 49.10 亿分红（体量感）
- 百分比数据可视化：99.78% vs 77.61% 的鲜明对比
- 勾选框 / 投票勾选元素
- 现金流箭头、信封（分红派息）
- 净利润下滑的下行小箭头 + 分红上行的对比，体现"业绩降但分红升"

## 构图方向
- 中心：电梯场景内嵌广告屏（分众传媒的业务符号）
- 左/上：分红主题区——大金币堆 + "49.10亿" + "166.66%" 醒目数据
- 右/下：议案对比区——两个投票勾选框，分别标注 99.78% 和 77.61%，形成视觉对比
- 顶部标题栏
- 边角点缀参会人数、股份占比等小数据标签
