# 内容分析 - 马斯克万亿美元身家 / SpaceX 上市

## 主题
SpaceX 创纪录 IPO + 马斯克成为人类首位万亿美元富翁，核心张力是"摇滚式敲钟"与"给人类未来定价"。

## 关键信息点
- **IPO 数据**: 6月12日纳斯达克挂牌，代码 SPCX，发行价 135 美元，募资 **750亿美元**（刷新全球IPO纪录）
- **首日表现**: 涨 19.22%，市值约 **2.11万亿美元**
- **敲钟场景**: 得州星际基地 Starbase，星舰原型机矗立，马斯克 + 标志性喷火吉他，摇滚气质
- **马斯克语录**: "带你们去月球，去火星，去更远的地方"
- **万亿美元身家构成**:
  - SpaceX 持股 41-42% → 约 **8700亿美元**
  - 特斯拉持股 → 超 **2600亿美元**
  - 合计正式跨过 **1万亿美元** 门槛
  - 同时掌控两家万亿美元级公司
- **估值争议**: 2025年营收约187亿、净亏损49亿，市值/营收 = **110倍**，传统框架难解释
- **定价哲学**: 不是给利润定价，而是给"人类未来路径"定价
