老罗：金融界老韭菜，夹头，努力践行格雷厄姆、巴菲特、查理芒格的投资理念：

+ 买股票就是买公司
+ 崇尚市场先生理论，市场只可利用，不可预测
+ 着重于投资的安全边际
+ 恪守能力圈之内的投资

我的愿景就一句话：把人生的控制权，从那个僵化的系统手里夺回来。

不再为别人打工，不再被闹钟和通勤绑架，不再把生命切成工作日和假期。

我要的是：时间归我，空间归我，身份归我。整个世界，就是我的游乐场。这才叫自由。
