主题：2026年5月社零同比-0.6%
内容：

5月社零数据出来那天晚上，我正好在翻手机。-0.6%，第一反应是：这么差？仔细看完之后，我踏实了，接着等。

拖后腿的主力只有一个：汽车。去掉汽车类之后，社零增速变成了正的1.1%。换句话说，汽车一个人把整个大盘拖成了负数。汽车类零售额同比增速1.1%，比整体低了1.7个百分点。这不是本月才出现的情况，4月的汽车增速就已经很弱，5月只是没有继续恶化。为什么卖不动？前两年国补力度太猛了。新能源车购置税减免、以旧换新补贴、地方消费券，一套组合拳把购车需求集中释放。现在政策退坡，自然没人接盘。另一方面，燃油车份额还在萎缩，今年1-5月燃油车销量同比下滑超过15%。电动车这边价格战从2024年打到2026年，还没打完，消费者的心态变成了再等等，还能更便宜。这不是不买，是持币观望。

所以整体看下来，消费没崩，是钱没往车上花。如果所有品类都在跌，那是系统性问题；但如果只有一个大品类在跌，其他品类正常甚至上涨，那只是结构性问题。

把各品类增速拉出来看，分化的痕迹很清晰。涨的分项：饮料6.1%、烟酒4.8%、服装鞋帽3.8%、化妆品2.5%、粮油食品5.6%、体育娱乐用品8.2%。跌的分项：家电-15.6%、家具-8.7%、金银珠宝-8.9%。建材这个品类5月数据是正的5.8%，但主要靠基建和工程端撑着，居民端的装修需求其实很弱。跟房子有关的，除了建材在硬撑，基本都趴着；跟让自己舒服点有关的，全在涨。家电跌了15.6%，一方面是去年以旧换新政策把今年的需求提前消耗掉了，另一方面是房子卖不动，装修需求自然也少。家电和家具都是典型的后周期品类，先有房才有家电，房子不卖，这些就起不来。而饮料、烟酒、服装、化妆品、体育用品，都是即时满足和日常享受，花小钱让自己高兴的东西。大家不买电视了，但奶茶照喝、烟酒照买、出门还得穿得体面点。

钱从家里用的搬到了自己用的。与其说消费降级，不如说消费在转移。做地产链生意的还得熬，做饮料、美妆、户外、体育用品的，别被宏观标题吓到，数字摆在这儿。

还有一个细节。老有人说年轻人没钱了，消费起不来。前半句对，后半句我存疑。年轻人收入确实偏弱，高校毕业生规模还在创新高，就业市场的供需没有根本性改善，AI对白领岗位的替代效应还在发酵。但消费的主力盘可能根本不在年轻人手里。谁手里有钱？50到65岁这批刚退休或正在退休边缘的人。有房，大概率没有房贷了；有车，该换的也换过了；有积蓄，退休金不低于在职时的七八成；身体还行，心态比实际年龄年轻十岁；子女大部分已经独立，不需要他们贴补了。这批人的消费画像和年轻人完全不同。他们不太买潮玩，但会买高品质的食品和酒水；不太追新品牌，但认老牌子；不太冲动消费，但该花的一分不少。如果市场还在用年轻人的消费力来推断整体趋势，可能是看错了方向。
